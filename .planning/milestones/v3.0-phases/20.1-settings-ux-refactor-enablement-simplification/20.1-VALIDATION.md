## Sampling Rate

- After every renderer settings change: run the quick renderer Settings/LLMSetup command.
- After cross-tier audio changes: run renderer, electron IPC, and sidecar focused suites.
- Before verification: run renderer typecheck, contract codegen idempotence, and stale-gate search.
- Max feedback latency: ~90 seconds for focused validation.
