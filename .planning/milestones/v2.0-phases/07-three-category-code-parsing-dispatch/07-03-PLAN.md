In `tts_preprocessor.py`, add `ignore_curly_brackets: bool = True` to config
and filtering calls, and add `filter_curly_brackets(text)` implemented through
the existing nested-filter helper with `{` and `}`.

In `transformers.py`, remove `AvatarCapabilities`, `ActionIntent`,
`actions_extractor`, and `_extract_intents`. Import `ActionCode`, `Dispatch`,
`EventFire`, `VariantToggle`, `EventEntry`, and `VariantEntry` from contracts.

Add `_variant_lookup(variants)` and `_event_lookup(events)` helpers keyed by
lowercase `entry.code`.

Add `code_extractor(plugin_action_codes, variants, events)` as a decorator
factory. It must case-fold plugin action codes and catalog codes once when the
decorator is created. Implement `_extract_dispatches(text, plugin_action_codes,
variants_by_code, events_by_code) -> list[Dispatch]` as one left-to-right
while-loop over lowercase text with opener map
`{"[": ("]", "action"), "{": ("}", "variant"), "<": (">", "event")}`.

For `[name]`, emit `ActionCode(name=name)` only when the lowercased name is in
plugin action codes. For `{name}`, emit `VariantToggle(name=name,
hotkey_id=entry.hotkey_id)` only when the lowercased name is in the variant
lookup. For `<name>`, emit `EventFire(name=name, hotkey_id=entry.hotkey_id,
duration_ms=<computed_delay_ms>)` only when the name is in the event lookup.
Compute `<computed_delay_ms>` from `EventEntry` exactly once: if
`entry.duration_is_fallback` is true, use `10000`; otherwise use
`max(0, int(entry.duration_seconds * 1000)) + 1000`. Unknown or mismatched tags
are silently dropped.

`code_extractor` must yield `(SentenceWithTags, dispatches)` and must not mutate
`sentence.text`, so the active plugin still sees original bracketed text per
D-A2.

Update `display_processor()` and `tts_filter()` to pass `dispatches` through and
to strip square, curly, and angle syntaxes. Do not add a `<think>` strip step;
PARSE-04 is covered by unknown-event drop plus boot reserved-name validation.

Move tests from `sidecar/tests/test_actions_extractor.py` into
`sidecar/tests/orchestrator/test_code_extractor.py`, delete the old file, and
add the 16 fixtures from 07-RESEARCH Section C.
