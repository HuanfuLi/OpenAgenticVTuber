<objective>
Add VTS cursor head-center calibration as a Phase 13 side feature.

Purpose: the current cursor tracker projects around the VTS window center, which drifts when the user grabs and moves or resizes the model inside VTube Studio. Add a manual recompute button and an automatic recompute toggle so the user can re-center cursor tracking around the current model head/center while keeping the Phase 10 sidecar-owned cursor pipeline intact.

Output: sidecar cursor calibration state and admin endpoints, calibrated `CursorDriver` projection, Electron/preload bridge, Settings > VTube Studio controls, regression tests, and UAT notes.
