- `class ManifestChangeWatcher` wrapping a watchdog `Observer`.
- `def start_manifest_change_watcher(manifest_path: Path, boot_manifest: PluginManifest) -> ManifestChangeWatcher`.

call `build_action_codes_section()` from the watcher.

- unrelated file path is ignored.
- invalid YAML logs `[PLUGIN-MANIFEST-WATCH]` and does not raise.
- source inspection asserts the watcher helper does not call
  `build_action_codes_section`.
