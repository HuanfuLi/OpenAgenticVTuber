<objective>
Close the two core Phase 6 runtime behavior gaps from `06-VERIFICATION.md`:
preserve bracketed action-code text into the plugin adapter, and make `[joy]`
produce timed nonzero ParamFrame ramp frames through runtime ticks.

Purpose: The default plugin exists, but the normal LLM path strips `[joy]`
before plugin delivery and the adapter currently holds the all-zero activation
frame. This plan restores the plugin data flow required by PLG-07, ARCH-03, and
ARCH-04 without reintroducing direct pyvts or exp3 behavior.

Output: Focused regression tests and implementation for plugin-visible raw
sentence text plus timed action-frame rendering.
</objective>
