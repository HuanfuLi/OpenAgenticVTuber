    Create `sidecar/src/sidecar/vts/speech_mouth_driver.py` with `class SpeechMouthDriver`:
    - constructor signature `SpeechMouthDriver(writer: ParameterWriter, now: Callable[[], float], tick_seconds: float = 1 / 60, mouth_param_id: str = "ParamMouthOpenY")`
    - `async def consume_forever(self, queue: asyncio.Queue[SpeechEnvelopePayload]) -> None`
    - `async def drive_envelope(self, envelope: SpeechEnvelopePayload) -> None`
    - `async def _write_mouth(self, value: float) -> None`

    In `drive_envelope`, compute mouth values from the real envelope per D-02/D-05:
    - `elapsed_ms = max(0.0, (self._now() - envelope.started_at) * 1000.0)`
    - `position = elapsed_ms / envelope.slice_length`
    - `index = int(position)`
    - interpolate between `volumes[index]` and `volumes[index + 1]` when both exist
    - clamp to `0.0..1.0`
    - call `await self._write_mouth(interpolated_value)` each tick
    - after the envelope is exhausted, call `await self._write_mouth(0.0)` exactly once

    `consume_forever` must `await queue.get()`, `await drive_envelope(envelope)`, and `queue.task_done()` in a `finally`.
