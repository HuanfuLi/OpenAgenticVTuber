## Cross-Phase Integration

The core walking-skeleton chain is verified:

1. Electron launches the sidecar and receives the ready WebSocket endpoint.
2. Renderer setup gates LLM config and restarts the sidecar when first-time config is saved.
3. LLM output streams through the sentence divider and tag extractor.
4. TTS synthesizes sentence-buffered audio and exposes RMS envelopes.
5. The compositor consumes RMS and intent events, then writes to VTube Studio through the single safe pyvts writer.
6. Live UAT confirms idle motion, speech motion, joy blend, cursor tracking, discrete hotkey demo, and the supported head_only body-motion fallback.
